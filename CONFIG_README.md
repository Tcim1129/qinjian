# 亲健 - 微信小程序配置文件模板

## 使用方法

1. 复制 `project.config.template.json` 为 `project.config.json`
2. 将 `YOUR_APPID_HERE` 替换为您的微信小程序 AppID
3. 复制 `miniprogram/project.config.template.json` 为 `miniprogram/project.config.json` 并同样替换 AppID

## ⚠️ 安全提醒

**不要将包含真实 AppID 的配置文件提交到 Git！**

文件 `project.config.json` 和 `project.private.config.json` 已添加到 `.gitignore`。

## 获取 AppID

1. 登录[微信公众平台](https://mp.weixin.qq.com/)
2. 进入：开发 → 开发管理 → 开发设置
3. 找到 AppID(小程序ID)
