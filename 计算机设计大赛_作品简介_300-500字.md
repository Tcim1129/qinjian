# 大学生计算机设计大赛作品简介（300-500字）

## 推荐版本

亲健是一套面向青年关系场景的 Web 关系支持系统，定位于 `软件应用与开发 > Web应用与开发` 赛道。项目聚焦情侣、夫妻、挚友等高频关系场景中普遍存在的沟通困难、误解积累和修复无抓手问题，目标不是提供泛聊天陪伴，也不是虚拟陪伴，而是通过完整的软件系统为真实关系提供持续支持。

系统围绕“记录、识别、干预、修复、回流”构建主链路。用户先完成每日记录，系统再结合关系画像、事件流和当前风险状态生成关系简报，并给出下一步更适合的行动方向。在真实沟通前，用户可以使用消息预演功能降低表达风险；在双方叙事发生明显错位时，系统可生成双视角对齐结果；当关系进入高风险阶段时，系统进一步输出结构化修复协议。所有关键动作都会进入时间轴与证据抽屉，便于回放、解释和复盘。

项目采用 Web 前端、FastAPI 后端和 PostgreSQL 数据层构成完整软件架构，并在此基础上形成记录、评估、干预、修复和回看的完整闭环。其创新不在于单纯接入某个模型，而在于把关系画像、关系事件流、预演、对齐和修复能力整合进同一套可解释、可追踪、可演示的系统。亲健希望将抽象的情绪和关系问题转化为软件能够持续支持的日常行动，体现计算机技术在青年关系支持场景中的实际应用价值。

## 100 字压缩版

亲健是一套面向青年关系场景的 Web 关系智能支持系统，通过关系记录、画像识别、消息预演、双视角对齐、修复协议和时间轴证据抽屉，形成可解释、可追踪、可持续的关系支持闭环。项目强调完整软件系统与真实场景落地，适合软件应用与开发赛道。
